How to Use

  • Make/Model: Enter exactly as shown on V5C logbook or vehicle registration.
  • Engine code: Optional, but helps match specific entries in the Annex (e.g., "1.4 TSI", "2.0 Duratec").
  • First use date: The date the vehicle was first registered. This determines which emission standards apply.
  • Fuel type: Petrol, diesel, gas (LPG/CNG), hybrid, or electric.
  • Catalytic converter: Check if the vehicle has one (most post-1992 petrol cars do). If unsure, you can leave it unchecked and the system will infer from date and model.
  • DGW / Seats: Optional, but helps classify as passenger car vs light goods vehicle, which affects standards.

The system uses the UK MOT Inspection Manual ( Passenger and Light Commercial Vehicles) plus the 19th edition emission standards tables.
