5-Gas Exhaust Analyzer

Embedded diagnostic tool for engine health assessment

How to Embed on Your Site

Add the following HTML snippet to your page. Replace YOUR_DOMAIN with the URL where this widget is hosted.

<iframe src="https://YOUR_DOMAIN/static/widget.html" width="100%" height="900" frameborder="0"></iframe>

Note: The widget must be served from the same origin as the analyzer for the sample data buttons to work. If embedding on a different domain, the buttons will not function due to browser security restrictions, but the analyzer will still be usable manually.
